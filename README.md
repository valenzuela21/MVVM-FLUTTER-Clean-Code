# MVVM Flutter Clean Architecture

This app is developed with Flutter and with the MVVM architecture and the Streammer State Manager.

## Database

SQlite and Sharepreferences.

## Getting Started

Before you start, follow these instructions:

```
flutter pub get
cd myProject
flutter run
```
